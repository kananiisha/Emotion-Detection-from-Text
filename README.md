# Emotion Detection from Text

A dark-themed Streamlit app for detecting the emotion of user-entered text. The app uses a simple NLP pipeline with TF-IDF vectorization and logistic regression, and includes an animated floating emoji background.

## What’s included

- `app.py` — Streamlit interface and emotion prediction logic
- `main.ipynb` — original notebook for model training and experimentation
- `train.txt`, `test.txt`, `val.txt` — dataset files used for training and evaluation
- `model.pkl`, `vectorizer.pkl` — serialized model and vectorizer generated by the app

## Features

- Clean, modern dark UI
- Floating emoji background animation
- Text preprocessing using lowercase, punctuation removal, and stopword filtering
- Emotion prediction on user input

## Setup

1. Create and activate a Python environment.
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```

2. Install dependencies.
   ```powershell
   pip install streamlit pandas scikit-learn nltk
   ```

3. Download NLTK stopwords if needed.
   ```python
   import nltk
   nltk.download('stopwords')
   ```

## Run the app

```powershell
streamlit run app.py
```

Open the local URL displayed by Streamlit in your browser.

## Notes

- `app.py` will train the model automatically if `model.pkl` and `vectorizer.pkl` are not present.
- The app uses `train.txt` as the default training dataset.
- The `.gitignore` excludes `model.pkl`, `vectorizer.pkl`, `.venv`, and `__pycache__`.
